WebMonkey

Configure custom JavaScript to run on every webpage

Script Execution

Enable custom script

When enabled, your custom script will run on all pages after they load.

Custom JavaScript

Script runs in page context with full DOM access.

Examples

// Remove all images
document.querySelectorAll('img').forEach(el => el.remove());
// Change all links to open in new tab
document.querySelectorAll('a').forEach(a => a.target = '_blank');
// Add a floating button
const btn = document.createElement('button');
btn.textContent = '🤖 Click Me';
btn.style.cssText = 'position:fixed;top:10px;right:10px;zIndex:9999;';
document.body.appendChild(btn);
// Highlight all links
document.querySelectorAll('a').forEach(a => {
  a.style.background = 'yellow';
});
